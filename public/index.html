帮助文档

快速开始

1. 导入文件

点击上方的“浏览文件”按钮或拖拽文件到上传区域，支持 XML、XLIFF、JSON、PO、RESX、iOS Strings 等多种格式。

2. 查看翻译项

文件解析后，翻译项将显示在中间面板。左侧为原文，右侧为译文。

3. 执行翻译

选中需要翻译的项，点击“翻译选中”或“翻译全部”按钮。

4. 导出结果

翻译完成后，点击右侧面板的“导出翻译”按钮，选择导出格式。

开发与部署

入口文件

应用入口为 public/index.html，在浏览器中打开该文件即可使用。

构建样式

样式由 Tailwind CSS 构建生成，输出到 public/styles.css。

常用命令：

  • npm run build-css：生成生产版本样式
  • npm run watch-css：开发时监听并自动重建
第三方库

第三方库资源位于 public/lib/，可通过 npm run update-cdn 更新本地文件。

部署

部署时只需要发布 public/ 目录中的文件。

常见问题

Q: 支持哪些文件格式？

A: 支持 XML、Android strings.xml、iOS Strings、JSON、XLIFF、PO、RESX 等多种格式。

Q: 如何配置 API 密钥？

A: 在右侧“翻译设置”面板中输入您的 API 密钥，系统会自动保存。

Q: 数据存储在哪里？

A: 所有数据都存储在您的浏览器本地，不会上传到服务器。
